# General Health Query Chatbot

## Objective

Build a chatbot that can answer general health-related questions using a Large Language Model (LLM) while encouraging safe and responsible use.

## Model

- Mistral-7B-Instruct-v0.2
- Hugging Face Inference API

## Features

- Answers common health questions
- Uses prompt engineering to control response style
- Includes safety checks for high-risk medical situations
- Interactive command-line chat interface
- Easy to run in Google Colab

## Example Questions

- What causes a sore throat?
- Is paracetamol safe for children?
- How can I reduce stress?
- What are the symptoms of dehydration?

## Safety Features

The chatbot avoids providing diagnoses, medication dosages, or emergency medical advice. Instead, it encourages users to consult qualified healthcare professionals when needed.
